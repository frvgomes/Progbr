- Cada tag é um elemento.
- w3schools.com é fonte oficial da documentação do html
- Tag meta é um exemplo de tag de auto-fechamento. Não precisa fechar a tag.
- Comentário no html <!-- termina em --> ( Pode usar atalho ctrl+;)
- Tag padrão xsemântica
    Negrito padrão <b> Semântica <strong>
    Itálico padrão <i> Semântica <em>
    Deletado padrão <d> Semântica <del>
    Sublinhado padrão <u> Semântica <ins>
- Tag img junto com atributo "alt" ajuda buscadores a reconhecer a imagem.

- A diferença entre id e class é que a class pode ser usada em mais de um elemento. 
o ID identifca um único elemento.
- ID pode ser usado para fazer um link dentro da própria página. "href=#identificador".
